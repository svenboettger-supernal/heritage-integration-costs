Heritage's vendor access costs

Recurring bills to Heritage

  • Yodlee (bank and brokerage aggregation): about $12-40K/yr at pilot scale, where the annual minimum dominates. Scales with users. Heritage holds the contract.
  • Property data (Zillow's API is retired): use an AVM. ATTOM is about $499/yr flat, or HouseCanary Pro at $790/yr plus $7 to $10 per report. Pilot cost is about $0.5 to $4K/yr.
  • Coinbase Prime: about $1,000 one-time, only if institutional custody is required. The retail read API is free.
  • Entra External ID: $0 under 50,000 monthly active users, then $0.03 per user, and only if Entra is used as a full identity store, which is not needed here.

Negotiated or unpublished: request a quote

  • Addepar: most likely $0, with a modest annual partner fee of about $10K to $30K as the downside. Pursue through a warm introduction to the partnerships team.
  • eMoney: $0 if read access rides the firm's license, otherwise a negotiated partner fee.
  • Carta: no API fee, but requires SOC 2 Type 2 and an invitation. The cost is compliance, not a license.
  • Zocks: the full REST API requires an Enterprise contract, priced on quote.
  • Jump: no self-serve API; a partnership is negotiated through business development, likely at $0 list.
  • Arch: fully quote-based, relevant only if clients hold private or alternative assets.
  • Salesforce: $0 for a private integration. Listing a managed app adds about $999 per app, $150 per year, and a 15% revenue share, none of which is required.
